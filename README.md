## ✨关于这个项目
欢迎大家关注微信公众号 **“水利研讨室”**。

**“水利研讨室”** 是一个关注水力学、流体力学 ~~扫盲~~ 科普的平台。

旗下主要包含【谁看不懂】和【谁做不出】两个栏目，前者侧重于基本原理及公式推导，后者侧重于计算流体力学实践。
本项目是【谁做不出】栏目的代码仓库，通过简单的代码实现一些流体力学问题的基本数值计算。
